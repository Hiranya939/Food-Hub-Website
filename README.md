🍔 Food Hub Website 🍟
The **Food Hub Website** is a modern web-based online food ordering system developed using HTML, CSS, and JavaScript. This project is designed to replicate the basic functionality of real-world food delivery platforms by providing a simple, interactive, and user-friendly interface where users can browse food items, manage their cart, and place orders efficiently.

The system is structured into multiple sections such as Home, Fast Food, Drinks, and Dessert. Each section is carefully designed to display food items in an organized and visually appealing way using card-based layouts. Users can easily navigate between these sections, explore available items, and interact with the system smoothly.

The Home page acts as the central hub of the application, featuring a welcoming hero section, highlighted features such as fast delivery, quality food, and easy ordering, as well as additional sections like delivery information and contact details. This provides users with a clear understanding of the platform’s purpose and usability.

## 🎯 Purpose of the Project

The main objective of this project is to demonstrate the practical application of front-end web development technologies while solving a real-world problem. It aims to build a digital food ordering interface that enhances user experience and simplifies the ordering process through an intuitive design and interactive features.

## ✨ Key Features

### 🏠 1. Home Page

* Attractive and modern UI design with gradient backgrounds
* Welcome section introducing the platform
* Feature highlights such as fast delivery, quality food, and customer support
* Easy navigation to different food categories

### 🍔 2. Fast Food Section

* Displays popular items such as burgers, pizzas, sandwiches, and fries
* Includes images, names, and pricing for each item
* Interactive buttons to add items to the cart

### 🥤 3. Drinks Section

* Shows beverages like juice, coffee, milkshakes, and soft drinks
* Clean and structured layout
* Easy selection and cart integration

### 🍰 4. Dessert Section

* Displays desserts like cakes, ice cream, brownies, and donuts
* Visually appealing card design
* Smooth browsing experience

### 🛒 5. Cart System

* Users can add multiple items to the cart
* Increase or decrease item quantity
* Automatic total price calculation
* Option to remove items or clear the cart
* Place order functionality with confirmation

### 📍 6. Delivery Address Section

* Allows users to enter delivery details such as name, address, and phone number
* Simulates real-world food delivery process

### 📞 7. Contact Section

* Displays contact information such as phone number and email
* Helps users connect with the service easily

## 🎨 User Interface Design

* Clean and modern layout with attractive color gradients
* Consistent design across all sections
* Card-based UI for better visualization of food items
* Smooth navigation and user-friendly interaction
* Responsive structure adaptable for different screen sizes (basic level)

## 🛠️ Technologies Used
